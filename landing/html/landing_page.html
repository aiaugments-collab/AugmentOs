Frequently Asked Questions

What makes BrowserOS different from other browsers?

BrowserOS is the first open-source browser with built-in AI agents. Unlike Chrome or Safari, BrowserOS prioritizes privacy and automation, letting you automate repetitive tasks using just natural language — "scrap prices from this page and add it to a spreadsheet "—   no coding needed.

What AI models does BrowserOS support?

BrowserOS supports OpenAI GPT models, Anthropic Claude, Google Gemini, and local models through Ollama and LMStudio. You can bring your own API keys and easily switch between providers.

What operating systems does BrowserOS support?

BrowserOS currently supports macOS, Windows, and Linux. The browser runs on the same system requirements as Google Chrome or Safari.

Is BrowserOS compatible with Chrome extensions?

Yes! BrowserOS is a Chromium fork, so all your favorite Chrome extensions work seamlessly. You can import your existing bookmarks, passwords, and extensions from Chrome with just a few clicks.

How much does BrowserOS cost?

BrowserOS is completely free and open source. You only pay for AI API usage if you choose to use cloud providers like OpenAI or Anthropic. Local models with Ollama and LMStudio are completely free.

Can I contribute to BrowserOS development?

Yes! We 're fully open source. Jump into our GitHub repo, Discord community , or send us feature requests. We actually read and respond to everything.
